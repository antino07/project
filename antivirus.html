Installation Steps

  1. 1

    For Windows Security (Built-in)

    Click Start, type "Windows Security" and open it. All protections are enabled by default in Windows 10/11.

    Alternative path: Settings → Update & Security → Windows Security

  2. 2

    Verify Protection Status

    Check that all protection areas show green checkmarks. Virus & threat protection should be active.

    Important: Ensure real-time protection is turned on for continuous security.

  3. 3

    Run a Scan

    Click "Virus & threat protection" then "Quick scan" or "Full scan" to check your system for threats.

    Schedule scans: Set up automatic weekly scans for ongoing protection.

  4. 4

    Third-Party Antivirus

    Download from official site (like avast.com). Run the installer and follow the setup wizard.

    Free options: Avast Free, AVG Free, Bitdefender Free, Malwarebytes Free

  5. 5

    Complete Installation

    Restart computer if prompted. Run first scan after installation and schedule regular scans.

    Update regularly: Keep virus definitions updated for maximum protection.

Popular Antivirus Software

  • Windows Security - Free, built into Windows 10/11
  • Avast Free Antivirus - Comprehensive free protection
  • Bitdefender Free - Lightweight and effective
  • Malwarebytes - Excellent for malware removal
  • Kaspersky Free - Strong detection rates

Important Warning

DO NOT install multiple antivirus programs simultaneously. They will conflict with each other and cause system instability. Choose either Windows Security OR one third-party antivirus.

If installing third-party antivirus, Windows Security will automatically disable itself.
